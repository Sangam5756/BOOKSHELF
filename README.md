## Setup Instructions

1. Clone the repository:
    ```bash
    git clone <https://github.com/Sangam5756/BOOKSHELF.git>
    cd BOOKSHELF
    ```

2. Install dependencies:
    ```bash
    npm install
    ```

3. Run the application:
    ```bash
    npm run dev
    ```
